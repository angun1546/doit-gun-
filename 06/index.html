HTML의 기본문법

HTML문서는 웹페이지의 뼈대 역할을 함
계층구조를 가진다

태그의 사용법

태그는 여는 태그와 닫는태그로 구성됨.
태그의 표시는 <> 괄호를 사용한다. <태그></태그>

태그를 표시하는 기호는 < > 이며 괄호내에 명령어를 넣어 사용한다. 공백을 표시할때는            를 사용한다.

태그의 분류

태그가 웹페이지에서 차지하는 영역에 따라 두가지로 나눌 수 있다.

분류 설명 특징 태그 예시
블록레벨
(Block Level) 		웹페이지의 화면 전체 너비를 차지 • 새로운 줄에서 시작
• 전체 너비를 차지
• 위아래 마진 적용됨
• 가로/세로 크기 조정 가능 		<h1>~<h6>
<p>
<div>
<ul>, <ol>
<li>
<section>
<header>, <footer>
인라인레벨
(Inline Level) 		웹페이지에서 콘텐츠의 너비만큼 차지 • 같은 줄에 배치
• 콘텐츠 크기만 차지
• 좌우 마진만 적용됨
• 가로/세로 크기 조정 불가 		<a>
<span>
<strong>, <em>
<img>
<button>
<input>
<label>

블록 레벨 태그 상세

태그 설명 사용 예시
<h1>~<h6> 제목 태그 (h1이 가장 큼) <h1>주제</h1>
<p> 단락 (Paragraph) <p>본문 내용</p>
<div> 콘텐츠 분할 용도 <div>내용</div>
<ul>, <ol> 비순서/순서 리스트 <ul><li>항목</li></ul>
<table> 테이블 (표) <table><tr><td>셀</td></tr></table>

인라인 레벨 태그 상세

태그 설명 사용 예시
<a> 하이퍼링크 <a href="url">링크</a>
<span> 인라인 콘텐츠 그룹 <span>텍스트</span>
<strong>, <em> 강조 텍스트 <strong>굵게</strong>, <em>이탤릭</em>
<img> 이미지 <img src="파일" alt="설명">
<button> 버튼 <button>클릭</button>
